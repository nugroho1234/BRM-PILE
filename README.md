# PT. BRM PILE - Official Website

Official website for PT. Balikpapan Ready Mix Pile, specializing in high-quality prestressed precast concrete products.

## Quick Start

### View the Website

The ready-to-deploy website files are located in the **`dist/`** folder.

- Open `dist/index.html` in your browser to view the website locally
- Or deploy the entire `dist/` folder to your web server

### Development

To make changes to the website:

1. Edit files in the `src/` folder:
   - `src/index.html` - Main HTML file
   - `src/styles.css` - All styling
   - `src/script.js` - Interactive features

2. Run the build script to update the `dist/` folder:
   ```bash
   ./build.sh
   ```

3. Preview during development:
   ```bash
   npm run dev
   # or
   python3 -m http.server 8001 --directory src
   ```

## Features

- **Responsive Design**: Fully responsive across desktop, tablet, and mobile devices
- **Modern UI**: Clean, professional design with Poppins font
- **Smooth Navigation**: Smooth scrolling between sections
- **Product Showcase**: Complete product catalog with images
- **Contact Form**: Ready for Web3Forms integration
- **WhatsApp Integration**: Floating WhatsApp button for easy contact
- **ISO Certifications**: Highlighted certifications section
